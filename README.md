# Органайзер скарба

Серверная часть системы учета награбленного

# Модель
## Пользователи

Модель пользователя определена в классе [User](src/main/java/ru/homestorage/model/User.java).
Регистрация пользователей осуществляется по email. Подтверждение email будет реализовано в дальнейшем.

При регистрации и авторизации возвращается JWT для доступа к ресурсам приложения.

## Контейнер

Контейнер [Container](src/main/java/ru/homestorage/model/Container.java) представляет собой универсальную сущность для хранения места расположения вещи.
Контейнеры могут быть вложены друг в друга. Типы контейнеров описаны в [ContainerType](src/main/java/ru/homestorage/model/enums/ContainerType.java): здание, помещение, мебель, полка, коробка...

Каждый контейнер имеет:
- владельца (ID пользователя)
- уровень доступа
- наименование
- тип
- дату создания
 
Также могут быть указаны:
- ID группы пользователей, имеющих доступ к контейнеру
- описание
- ID родительского контейнера
- QR-код
- дата изменения

## Группа

Группа [Group](src/main/java/ru/homestorage/model/Group.java) это группа пользователей, имеющих одновременный одинаковый доступ к контейнерам.

Для хранения состава группы используется сущность [GroupMember](src/main/java/ru/homestorage/model/GroupMember.java).
Она хранит ID группы, ID пользователя и роль пользователя в группе.

Роль в группе [GroupRole](src/main/java/ru/homestorage/model/enums/GroupRole.java):
- OWNER - Создатель группы
- ADMIN - Может управлять участниками
- MEMBER - Обычный участник

## Аудит 

Сущность [AuditLog](src/main/java/ru/homestorage/model/AuditLog.java) предназначена для хранения истории действий над контейнерами и вещами.
